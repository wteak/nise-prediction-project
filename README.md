# ☀️ 대용량 시계열 데이터 기반 태양광 일사량 예측 모델

## 1. 프로젝트 개요
본 프로젝트는 약 **1,900만 행(2.5GB)**에 달하는 방대한 환경 센서 데이터를 분석하여 태양광 발전의 핵심 변수인 **'일사량(nins)'**을 예측하는 AI 모델링 프로젝트입니다.

전체 데이터의 **90% 이상이 결측치**로 이루어진 극한의 데이터 환경에서, 단순한 모델 튜닝을 넘어 **데이터 중심(Data-Centric)**의 전처리 전략과 **물리적 도메인 지식(Physics-Informed)**을 결합하여 결측 정보를 복원하고 예측 정확도를 극대화하는 데 초점을 맞췄습니다.

* **활동 기간:** 2025.10.25 ~ 2025.11.14
* **참가 형태:** 팀 프로젝트 (우택 외 1명)
* **주요 목표:** 대용량 시계열 데이터 파이프라인 구축 및 결측치 복원 최적화

## 4. 핵심 수행 내용 (Key Process)

### 1) 데이터 전처리 최적화
* **메모리 효율화:** 초기 분할 처리(Chunking) 방식에서 발생하는 병목 현상을 해결하기 위해 물리 메모리를 증설하고, 입출력 속도가 빠른 Feather 포맷을 도입하여 데이터 로딩 속도를 획기적으로 개선했습니다. 
* **보간 전략 수립:** 선형이나 스플라인 보간은 긴 결측 구간에서 왜곡된 패턴(Artifacts)을 생성함을 확인했습니다. [cite_start]이에 물리적 추세를 유지하면서 노이즈를 최소화하는 FFill(Forward Fill)과 BFill 방식을 최종 채택했습니다.

### 2) 통합 파이프라인 구축 (One-Universe)
* **문제 정의:** 학습 데이터(Train)와 테스트 데이터(Test)를 개별 처리할 경우, 군집화 기준이 달라지거나 시계열 연결성이 끊겨 예측 성능이 하락하는 분포 불일치 문제가 발생했습니다.
* **해결 방안:** 두 데이터를 하나로 합쳐 처리하는 **'One-Universe 파이프라인'**을 구축하고, 처리 후 다시 분리할 때 **원본 순서(Index)를 100% 복구하는 안전장치**를 마련하여 데이터 정합성을 확보했습니다. 
### 3) 이중 군집화 전략 (Dual Clustering)
단순 군집화의 한계를 넘기 위해 데이터 특성을 이원화하여 반영했습니다. 
* **위치 군집 (Location, K=20):** 위도와 경도 좌표를 기반으로 미세한 지형지물(산, 건물 그늘) 특성을 반영하기 위해 세분화했습니다.
* **기후 군집 (Climate, K=10):** 기온, 습도, 풍속을 기반으로 지리적 위치와 무관하게 유사한 기상 패턴을 공유하는 발전소들을 묶어 일반화 성능을 강화했습니다.

### 4) 물리 도메인 지식 적용 (Physics-Informed)
* **시차 문제 해결:** 데이터의 타임스탬프(UTC)와 한국의 실제 일조 시간(KST) 간의 **9시간 시차**로 인해 발생한 물리적 불일치(밤에 발전량이 높은 오류)를 규명하고, 이를 데이터 로드 단계에서 보정했습니다.
* **태양 고도 변수:** 단순 시간 변수 대신, 태양의 남중 고도와 계절별 낮의 길이 변화를 반영한 **코사인(Cosine) 변수**를 생성하여 모델이 '낮과 밤의 물리적 주기'를 이해하도록 설계했습니다.
